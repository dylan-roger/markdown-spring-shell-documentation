    Commands guide

    User commands

    create-user

    Create a user and returns its API key.

    Name Required Default Value Description
    -n, --name true The name of the user to add.
    -d, --duration false 365 The number of days before the API key expires.

    delete-user

    Delete a user from the system knowing its API key.

    Name Required Default Value Description
    --api-key true The API key of the user to delete.

    renew-user

    Add time to live to the API key of a specific user.

    Name Required Default Value Description
    --api-key true The API key of the user to renew.
    -d, --duration false 365 The number of days to live to add to this API key.
